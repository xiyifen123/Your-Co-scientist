# Your Co-scientist

AI Copilot for Wet Lab Scientists.
This is designed from real wet-lab research workflows to help scientists focus on discovery rather than protocol management.

## Product Vision

Your Co-scientist helps researchers execute experiments through adaptive workflow, voice interaction and intelligent reminders.

## Problem

Wet lab experiments involve complex protocols, frequent manual checking and potential operation errors.

## Core Features

- AI Adaptive Workflow
- Voice-first Interaction
- Smart Timer
- AI Calculator
- Personal Error Library

This project includes:
